# 🌿 Sheun Georrel Tabuzo — Personal Portfolio Website

> Live Site: **[https://g300000.github.io](https://g300000.github.io)**

A personal portfolio website built from scratch using pure **HTML**, **CSS**, and **JavaScript** — no frameworks, no libraries. Designed with a unique *Forest × Tech × Game* aesthetic inspired by my love for nature, coding, and game development.

---

## 🚀 Features

- **Responsive Design** — works on desktop, tablet, and mobile
- **Smooth Scroll Navigation** — fixed nav with active link highlighting
- **Animated Progress Bars** — triggered on scroll into view
- **Scroll Reveal Animations** — staggered card entrance effects
- **Typing Effect** — animated greeting on the hero section
- **Clickable Project Overlays** — hover project images to go directly to GitHub repos
- **Contact Form Validation** — JavaScript form validation with alert feedback
- **Mobile Menu Toggle** — hamburger menu for smaller screens
- **Unique Visual Identity** — hex-grid background, outlined hero name, bracket nav effect, corner card accents, gold achievement shimmer

## 💻 How to Run Locally

1. Clone or download this repository
2. Open the folder in **VS Code**
3. Install the **Live Server** extension *(by Ritwick Dey)*
4. Right-click `index.html` → **"Open with Live Server"**
5. View at `http://127.0.0.1:5500`
